## 第二步：运行升级后的 test.py

执行：

```bash
python3 test.py
```

你会看到一个菜单：

```text
====== 功能菜单 ======
1. 显示欢迎信息
2. 计算两个数字的和
3. 退出程序
```

你可以反复输入 `1`、`2`、`3` 来选择功能：

- 输入 `1`：输入你的名字，程序会打印欢迎语。
- 输入 `2`：输入两个数字，程序会帮你相加。
- 输入 `3`：退出程序。

## test.py 学到了什么

- `while True`：让程序循环运行。
- `if / elif / else`：根据你的选择执行不同功能。
- `input()`：接收用户输入。
- `float()`：把输入的文本转换成数字。
- `try / except`：输入不是数字时，给出友好提示，不让程序崩溃。
